# SID Tune Maker

Retro C64-style music generator. Make 8-bit bangers with drag & drop.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Features

- Drag & drop sequencer
- Authentic SID chip emulation (3 voices, filters, ring mod)
- Works on phone and desktop
- Works offline (PWA)
- Describe sounds in plain English (AI-assisted)
